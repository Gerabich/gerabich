Proposito del Software

El propósito de Visual Studio Code (VS Code) es proporcionar un entorno de desarrollo integrado (IDE) liviano y altamente personalizable para programadores. VS Code es desarrollado por Microsoft y se ha vuelto extremadamente popular debido a su flexibilidad y capacidad para admitir una amplia gama de lenguajes de programación.

Proceso de Instalación

Para instalar Visual Studio Code, deber`as:

  1. Descargar Visual Studio Code -> Descargar
  2. Abrir archivo de instalacion, seguir instrucciones (dar siguiente)
  3. Abrir VSC
  4. Personalizar con las extenciones necesarias
  5. Listo, ahora a trabajar con VSC
